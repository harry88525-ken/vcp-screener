結論先行

台虹(8039)尚未突破，標「觀察」。目前收148元，距樞紐高157.5元約差6%，緊縮結構已成形（pivot width 13.97%），RS 83分，族群排名第3，軟板材料AI/車用題材發酵，且近日外資轉買，具備補漲條件。

族群定位

台虹(8039)所屬「電子零組件業」族群，167檔成員中排名第3，屬於族群前段班（top）。族群近半年動能（126日）達+51.3%、3個月動能+33.4%，動能強勁，族群廣度0.82、VCP數量達119檔，屬於全市場動能數一數二的大族群。族群內RS最強成員包括華新科(2492)、禾伸堂(3026)、科嶠(4542)、信昌電(6173)、南電(8046)，RS皆達99分接近滿分，屬於族群超級龍頭；台虹(8039)以RS 83分排名第3，緊追在後，具備補漲想像空間。同業軟板材料廠如臻鼎-KY(4958)、台郡(6269)、亞電(4939)、達邁(3645)近期同樣受惠軟板旺季備貨題材，形成族群共伴效應。

基本面＋估值

月營收方面，2026年5月營收9.14億元，累計前5月營收約42.4億元；市場報導5月營收重返年增軌道，法人看好今年EPS年增率可望達45%左右。公司2026年營運規劃將FCCL、軟板材料業務從「追求營收規模」轉向「提高毛利率」，採嚴格篩選訂單方式優化產品組合。季度數據上，2026Q1 EPS 0.65元，優於去年同期2025Q1的0.32元，惟低於2025Q3的0.88元；毛利率方面2026Q1毛利5.32億元、營收24.15億元，毛利率約22%，較2024Q4曾出現營業虧損（-688萬元）已明顯改善。估值方面，PER 62.45倍，雖已從近1年高點78.9倍回落，但仍遠高於近1年低點12.8倍，屬於高評價成長股；PBR 3.68倍、殖利率僅1.34%，市場已給予顯著成長溢價。
